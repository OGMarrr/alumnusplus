## Run Locally

Clone the project

```bash
  git clone https://github.com/million23/alumnusplus.git
```

Go to the project directory

```bash
  cd alumnusplus
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm dev
```
